# 第6章 结论与展望

## 6.1 研究工作总结

本文针对航空餐食推荐场景中供给约束强、偏好动态变化和交互信号稀疏的特征，设计并实现了一个面向航空旅客的智能美食推荐系统，提出了Fused-PMFUP-PRMIDM-AMMBC三子模型融合的自适应混合推荐算法。主要工作包括：完成了航空餐食推荐场景的领域建模，涵盖航线-菜品绑定、舱位等级递进、多餐时序约束和库存限额等特殊约束，设计了18张数据表；提出了由PMFUP（内容匹配）、PRMIDM（漂移检测）和AMMBC（协同过滤）构成的自适应混合推荐算法，三子模型权重根据用户行为信号强度动态调整，并引入前序餐食惩罚机制（相同菜品×0.70、同类型×0.85）优化多餐推荐体验；基于Spring Boot 3 + Vue3 + UniApp技术栈构建了管理端、小程序端和后端服务三端系统，实现了偏好采集、智能推荐、预选确认、航后评分的完整闭环，共含65个REST API端点和两个定时任务；通过90名用户、10道菜品、4个航班的模拟数据集和29个测试用例进行了功能验证与算法效果分析，验证了三子模型的协同互补机制和系统的功能稳定性。

## 6.2 研究不足与展望

本文工作存在以下不足：（1）测试基于模拟数据，缺乏航空餐食领域的真实用户行为数据支撑；（2）基于用户的协同过滤在用户规模增大时计算复杂度较高；（3）超参数基于经验设定，缺乏系统化调优；（4）未引入航班时段、季节等上下文特征，偏好建模粒度有待细化；（5）缺乏精确率、召回率等离线评价指标的量化评估。未来工作可从以下方向展开：接入真实航空旅客数据进行算法验证与调优；引入近似最近邻搜索或基于物品的协同过滤优化计算效率；通过网格搜索或贝叶斯优化对超参数进行自动寻优；融合航班时段、航程时长等上下文特征提升偏好建模精度；建立包含离线指标计算和在线A/B实验的完整评价体系。
